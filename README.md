# Формат проверки

1. Из корня `task` запустите локальный сервер (любой статику). Примеры:
   - `python3 -m http.server 8000`
   - `npx serve` (если установлен)
2. Откройте страницу в браузере: `http://localhost:8000`.
3. Введите выражение из двух однозначных операндов и одного оператора (`+`, `-`, `*`, `/`). Пробелы можно ставить — они игнорируются.
4. Нажмите «Вычислить».
   - При ошибке увидите красное сообщение, инпут подсветится.
   - При успехе результат появится в блоке «result», без перезагрузки страницы.

## Файлы
- `index.html` — разметка и подключение стилей/скриптов.
- `styles/default.css` — базовые сбросы/фон.
- `styles/app.css` — оформление формы, подсказок и состояний.
- `utils.js` — валидатор и вычисление.
- `index.js` — логика формы и вывод сообщений.

## Примеры
- Валидные: `2+3`, `9/3`, `1-2`, ` 7 / 1`
- Ошибки: `1*` → «не хватает одного или нескольких операндов»; `1@2` → «в выражении должны использоваться только операторы +, -, /, *»; `3*g` → «операнды могут быть только числами».
